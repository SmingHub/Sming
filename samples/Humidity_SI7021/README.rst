SI7021 Humidity Sensor
======================

Introduction
------------

Example code for I2C SI7021 sensor board

This code uses the following GPIO:

-  #define I2C_SCL 5 // SCL
-  #define I2C_SDA 4 // SCA

Build instructions
------------------

Use ``make`` and ``make flash`` to build and flash the firmware to the
ESP8266 board.
