MCP23S17.h
1 /*
2  MCP23S17.h Version 0.1
3  Microchip MCP23S17 SPI I/O Expander Class for Arduino
4  Created by Cort Buffington & Keith Neufeld
5  March, 2011
6 
7  Features Implemented (by word and bit):
8  I/O Direction
9  Pull-up on/off
10  Input inversion
11  Output write
12  Input read
13 
14  Interrupt features are not implemented in this version
15  byte based (portA, portB) functions are not implemented in this version
16 
17  NOTE: Addresses below are only valid when IOCON.BANK=0 (register addressing mode)
18  This means one of the control register values can change register addresses!
19  The default values is 0, so that's how we're using it.
20 
21  All registers except ICON (0xA and 0xB) are paired as A/B for each 8-bit GPIO port.
22  Comments identify the port's name, and notes on how it is used.
23 
24  *THIS CLASS ENABLES THE ADDRESS PINS ON ALL CHIPS ON THE BUS WHEN THE FIRST CHIP OBJECT IS INSTANTIATED!
25 
26  USAGE: All Read/Write functions except wordWrite are implemented in two different ways.
27  Individual pin values are set by referencing "pin #" and On/Off, Input/Output or High/Low where
28  portA represents pins 0-7 and portB 8-15. So to set the most significant bit of portB, set pin # 15.
29  To Read/Write the values for the entire chip at once, a word mode is supported buy passing a
30  single argument to the function as 0x(portB)(portA). I/O mode Output is represented by 0.
31  The wordWrite function was to be used internally, but was made public for advanced users to have
32  direct and more efficient control by writing a value to a specific register pair.
33  */
